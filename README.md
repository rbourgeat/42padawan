## Requirements

- Install Docker

- Create your App on [42 intra](https://profile.intra.42.fr/oauth/applications/new), use `http://localhost:8000/auth/callback` as **Redirect URI** and fill the `.env.example`

## Usage

Up the project

```bash
make
```

Stop it

```
make down
```
